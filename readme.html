Foot Trace Processor

Convert foot traces to laser-cuttable SVG files for use with LightBurn.

Quick Installation

Run the install script:

chmod +x install.sh
./install.sh

Manual Installation

If you prefer to install manually:

  1. Install Homebrew: 
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. Install system dependencies: 
    brew install python@3.11
brew install opencv
brew install tcl-tk
  3. Install Python packages: 
    pip3 install --user opencv-python numpy Pillow svgwrite watchdog pyyaml

Running the Program

  1. Open Terminal
  2. Navigate to the program folder
  3. Run: python3 foot_trace_gui.py

Using the Program

  1. Click "Open Image" to load your foot trace photo
  2. Adjust the processing parameters:
    • Threshold: Controls how the black/white conversion happens
      Higher values detect lighter marks
    • Simplification: Controls how smooth the outline is
      Higher values create smoother but less detailed outlines
    • SVG Stroke Width: Controls the thickness of the line
      Keep this small (0.1-0.5) to avoid double-cutting in LightBurn
  3. Click "Generate SVG" to create the vector file
  4. The SVG will be saved in the same location as your input image

Tips for Best Results

  • Use a thick black marker for tracing
  • Place the paper on a contrasting surface
  • Ensure good lighting when taking the photo
  • Keep the camera parallel to the paper
  • Include the entire trace in the frame
  • For consistent sizing, always take photos from the same height

Troubleshooting

  • If no outline is detected:
    • Make sure the trace is dark and clear
    • Try increasing the threshold value
    • Ensure good lighting in the photo
  • If the outline is too detailed/jagged:
    • Increase the simplification value
    • Make sure the original trace is smooth
  • If LightBurn shows double lines:
    • Reduce the SVG stroke width

File Formats

  • Input: JPG, JPEG, or PNG images
  • Output: SVG files ready for LightBurn
